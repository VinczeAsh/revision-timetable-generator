# Revision Timetable Generator (VB.NET)

## Overview
A console-based application that generates structured weekly revision schedules up to an examination date.

## Features
- Topic allocation based on available study hours
- Balanced daily workload distribution
- Optional randomized topic scheduling
- Input validation to ensure accurate scheduling

## Technologies Used
- VB.NET
- Algorithmic scheduling
- Input validation
- Structured programming

## Purpose
This project was developed to strengthen my understanding of scheduling algorithms, structured logic, and workload optimisation.
